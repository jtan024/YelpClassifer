# YelpClassifer
a Tool to perform topic identification on Yelp reviews

## Dependencies:
1. numpy
2. pandas
3. re
4. nltk
5. spacy
6. gensim
7. sklearn
8. pyLDAvis
9. matplotlib
10. seaborn
(example of installation of package: **pip install numpy**)

**Run the following in terminal or command prompt**
python3 -m spacy download en
